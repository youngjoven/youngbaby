## 최종 아키텍처

```
[iPhone]
  ↓ HTTPS
[AWS Shield Standard]  ← 무료, 기본 DDoS 방어
  ↓
[AWS WAF]              ← Rate 30req/min, OWASP, IP Reputation
  ↓
[API Gateway]          ← Cognito JWT Authorizer
  ↓ AWS 내부 네트워크 (IGW/NAT 없음)
[Lambda - Private Subnet A/B (us-east-1a, us-east-1b)]
  ├── S3 ──────────── Gateway Endpoint  (무료)
  ├── DynamoDB ─────── Gateway Endpoint  (무료)
  ├── Bedrock ─────── Interface Endpoint (~$14.40/월, Multi-AZ)
  ├── SNS ────────────Interface Endpoint (~$14.40/월, Multi-AZ)
  ├── CloudWatch Logs  Interface Endpoint (~$14.40/월, Multi-AZ)
  └── EventBridge ─── Interface Endpoint (~$14.40/월, Multi-AZ)

VPC: 10.100.0.0/16
  ├── Private Subnet A: 10.100.1.0/24 (us-east-1a)
  └── Private Subnet B: 10.100.2.0/24 (us-east-1b)
```

## Phase 8. 배포 검증

```bash
# VPC 확인
aws ec2 describe-vpcs \
  --filters "Name=tag:Name,Values=youngbaby-vpc-dev" \
  --region us-east-1 \
  --query "Vpcs[0].{VpcId:VpcId,Cidr:CidrBlock}"

# VPC Endpoint 6개 확인 (Gateway 2 + Interface 4)
aws ec2 describe-vpc-endpoints \
  --region us-east-1 \
  --query "VpcEndpoints[].{Type:VpcEndpointType,Service:ServiceName,State:State}"

# Lambda LogGroup 8개 확인
aws logs describe-log-groups \
  --log-group-name-prefix /aws/lambda/youngbaby \
  --region us-east-1 \
  --query "logGroups[].{Name:logGroupName,Retention:retentionInDays}"
```

## 인프라 삭제 방법 (롤백)

```bash
cd backend/terraform/environments/dev

# DynamoDB는 prevent_destroy = true 설정으로 삭제 차단됨
# 삭제 전 dynamodb.tf에서 prevent_destroy = false 로 변경 필요
terraform destroy

# Bootstrap 리소스는 수동 삭제 (S3 버킷 비운 후)
aws s3 rm s3://youngbaby-terraform-state-ACCOUNT_ID --recursive
cd ../bootstrap && terraform destroy
```
